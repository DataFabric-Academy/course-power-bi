# Power BI Mastery Course

Knowledge Fabric: Academy-as-Code System

## Overview

This repository contains the Power BI Mastery Course documentation built with Docusaurus, following the Knowledge Fabric system architecture.

## Structure

```
course-power-bi/
├── website/          # Docusaurus website
│   ├── docs/        # Course documentation
│   ├── src/         # React components and pages
│   └── static/      # Static assets
└── README.md        # This file
```

## Development

### Prerequisites

- Node.js >= 20.0
- npm or yarn

### Getting Started

1. Navigate to the website directory:
```bash
cd website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run start
```

4. Build for production:
```bash
npm run build
```

## Deployment

This course is deployed via GitHub Actions to GitHub Pages under the `DataFabric-Academy` organization.
