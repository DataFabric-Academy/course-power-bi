# ยินดีต้อนรับสู่ Power BI Mastery Course

คอร์สเรียน Power BI แบบครบวงจรสำหรับผู้ที่ต้องการเชี่ยวชาญการใช้งาน Power BI ตั้งแต่ระดับพื้นฐานจนถึงขั้นสูง

## เกี่ยวกับคอร์สนี้

Power BI Mastery Course เป็นส่วนหนึ่งของ Knowledge Fabric: Academy-as-Code System ที่ออกแบบมาเพื่อให้คุณเรียนรู้ Power BI อย่างเป็นระบบและมีประสิทธิภาพ

### สิ่งที่คุณจะได้เรียนรู้

- **Power BI Fundamentals**: การใช้งาน Power BI Desktop, การเชื่อมต่อข้อมูล, การสร้างรายงานพื้นฐาน
- **DAX (Data Analysis Expressions)**: ภาษา DAX สำหรับการคำนวณและวิเคราะห์ข้อมูลขั้นสูง
- **Data Modeling**: การออกแบบ Data Model ที่มีประสิทธิภาพ, Relationships, และ Best Practices
- **Power Query (M Language)**: การทำ Data Transformation และ Data Preparation
- **Power BI Service**: การเผยแพร่รายงาน, Workspace Management, และการแชร์
- **Advanced Topics**: Performance Tuning, Security, Row-Level Security (RLS), และเทคนิคขั้นสูงอื่นๆ

## เริ่มต้นใช้งาน

### ความต้องการเบื้องต้น

- [Node.js](https://nodejs.org/en/download/) เวอร์ชัน 20.0 หรือสูงกว่า:
  - เมื่อติดตั้ง Node.js แนะนำให้เลือก checkbox ทั้งหมดที่เกี่ยวข้องกับ dependencies

### เริ่มต้นพัฒนา

1. ไปที่โฟลเดอร์ website:
```bash
cd website
```

2. ติดตั้ง dependencies:
```bash
npm install
```

3. เริ่ม development server:
```bash
npm run start
```

เว็บไซต์จะเปิดที่ http://localhost:3000 และจะ reload อัตโนมัติเมื่อคุณแก้ไขไฟล์

## โครงสร้างคอร์ส
